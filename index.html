造成边框变粗的原因

在 CSS 中的 1px 并不等于移动设备的 1px。这是由于不同的手机有不同的像素密度，在浏览器的 window 对象中有一个 devicePixelRatio 属性，可以反应 CSS 中像素与设备的像素比。

devicePixelRatio 的官方定义为：设备物理像素和设备独立像素的比例，也就是 devicePixelRatio = 物理像素 / 独立像素
